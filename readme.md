# shelbycorbitt.com

Personal website for Shelby Corbitt — a bio and links page hosted on GitHub Pages.

## About

A simple, single-page site built with plain HTML and CSS. No frameworks, no dependencies, no build tools required.

## Structure

```
index.html   # The entire site — all HTML, CSS, and content in one file
README.md    # This file
```

## Deploying

This site is hosted via **GitHub Pages**. Any change committed to the `main` branch is automatically published to [shelbycorbitt.com](https://shelbycorbitt.com) within a minute or two.

## Making changes

To update content (bio, links, etc.), edit `index.html` directly in the GitHub web editor and commit. No local setup needed.

## Custom domain

The custom domain `shelbycorbitt.com` is configured via DNS settings at the domain registrar and the GitHub Pages settings in this repository.
